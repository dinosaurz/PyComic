PyComic: Comic viewer
=====================

Simple comic viewer that currently has a text based interface for `XKCD <http://www.xkcd.com/>`_.

::
    >>> xkcd_handle = xkcd()
    ...
    >>> print xkcd_handle.comics
    ...
    >>> xkcd_handle.next()
    >>> xkcd_handle.previous()

Features
--------

- Quickly manage all images on the xkcd website
